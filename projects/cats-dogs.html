Project Summary

This project implements an image classifier that distinguishes between cats and dogs using convolutional neural networks (CNN). The model is trained with thousands of images and learns to identify distinctive visual features of each animal.

Development Process

1

Image Preprocessing

Pixel normalization (0-255 to 0-1), resizing to 150x150 pixels, and organization into training, validation, and test directories.

Normalized and resized images

2

Data Augmentation

Application of random transformations: rotation (40°), horizontal/vertical shift (20%), shear (20%), zoom (0.2), and horizontal flip to increase dataset variability.

Examples of images with data augmentation

3

CNN Architecture

Network construction with convolutional layers, MaxPooling, Dropout, and dense layers. The model learns hierarchical features from edges to complex shapes.

4

Training

Training for 15 epochs with batch size of 128. Monitoring accuracy and loss in training and validation to prevent overfitting.

Accuracy and loss graphs during training

5

Evaluation and Predictions

Evaluation on test set with 50 images. Visualization of predictions with confidence percentages for each image.

Grid of 50 test images with predictions

Results

Dataset

  • Training: 2000 images (1000 cats, 1000 dogs)
  • Validation: 1000 images (500 cats, 500 dogs)
  • Test: 50 images

Model

  • Architecture: CNN with multiple convolutional layers
  • Epochs: 15
  • Batch Size: 128
  • Data Augmentation: Yes

Features

  • Automatic image preprocessing
  • Data augmentation to improve generalization
  • Prediction visualization with confidence
  • Scalable model for more classes
