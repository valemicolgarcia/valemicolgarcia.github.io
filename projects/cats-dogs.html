Resumen del Proyecto

Este proyecto implementa un clasificador de imágenes que distingue entre gatos y perros usando redes neuronales convolucionales (CNN). El modelo se entrena con miles de imágenes y aprende a identificar características visuales distintivas de cada animal.

Proceso de Desarrollo

1

Preprocesamiento de Imágenes

Normalización de píxeles (0-255 a 0-1), redimensionamiento a 150x150 píxeles, y organización en directorios de entrenamiento, validación y prueba.

Imágenes normalizadas y redimensionadas

2

Data Augmentation

Aplicación de transformaciones aleatorias: rotación (40°), desplazamiento horizontal/vertical (20%), shear (20%), zoom (0.2), y flip horizontal para aumentar la variabilidad del dataset.

Ejemplos de imágenes con data augmentation

3

Arquitectura CNN

Construcción de la red con capas convolucionales, MaxPooling, Dropout y capas densas. El modelo aprende características jerárquicas desde bordes hasta formas complejas.

Input (150x150x3)
↓

↓

Output (1 neuron)
4

Entrenamiento

Entrenamiento durante 15 épocas con batch size de 128. Monitoreo de accuracy y loss en entrenamiento y validación para prevenir overfitting.

Gráficos de accuracy y loss durante el entrenamiento

5

Evaluación y Predicciones

Evaluación en conjunto de prueba con 50 imágenes. Visualización de predicciones con porcentajes de confianza para cada imagen.

Grid de 50 imágenes de prueba con predicciones

Resultados

Dataset

  • Entrenamiento: 2000 imágenes (1000 gatos, 1000 perros)
  • Validación: 1000 imágenes (500 gatos, 500 perros)
  • Prueba: 50 imágenes

Modelo

  • Arquitectura: CNN con múltiples capas convolucionales
  • Épocas: 15
  • Batch Size: 128
  • Data Augmentation: Sí

Características

  • Preprocesamiento automático de imágenes
  • Data augmentation para mejorar generalización
  • Visualización de predicciones con confianza
  • Modelo escalable para más clases
