# Certificates Folder

Place your certificate PDF files here with the following naming convention:

- `fce-2024.pdf` - First Certificate in English (FCE)
- `data-science-i-2024.pdf` - Data Science I: Fundamentals
- `machine-learning-python-2024.pdf` - Machine Learning with Python
- `data-science-ii-2024.pdf` - Data Science II: Machine Learning
- `data-science-iii-2024.pdf` - Data Science III: NLP & Deep Learning
- `cloud-computing-2024.pdf` - Cloud Computing (AWS)
- `sql-2025.pdf` - SQL
- `backend-i-2025.pdf` - Backend Programming I
- `backend-ii-2025.pdf` - Backend Programming II
- `toefl-2025.pdf` - TOEFL iBT

The download buttons on the website will automatically link to these files.
